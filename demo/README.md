# /demo — 구현 예정 기능 데모

이 폴더는 실제 서비스에 반영하기 전에 **UI/UX를 미리 체험**해보기 위한 데모 파일 모음입니다.  
Supabase 연결 없이 `localStorage`만으로 동작합니다.

---

## 파일 목록

| 파일 | 설명 | 상태 |
|------|------|------|
| `editor.html` | 입문서 에디터 데모 (어드민용) — Quill.js 기반 리치텍스트 에디터 | ✅ 완료 |
| `viewer.html` | 입문서 뷰어 데모 (학습자용) — 에디터 콘텐츠 렌더링 | ✅ 완료 |

---

## 체험 방법

1. `editor.html` 에서 내용 작성 → **저장하기** 클릭
2. **"학습자 화면으로 보기"** 버튼 → `viewer.html` 로 이동
3. 학습자 입장에서 페이지 넘기기, 북마크, 메모 등 체험
